# Top Papers — AI Agents & Embodied AI (2026-04-16)

## [TREX: Automating LLM Fine-tuning via Agent-Driven Tree-based Exploration](https://arxiv.org/abs/2604.14116)
**Authors:** et al. (Submitted April 15, 2026)
**TL;DR:** TREX is a multi-agent system that automates the full LLM training lifecycle by orchestrating a Researcher agent and an Executor agent. The fine-tuning process is modeled as a search tree, enabling the system to plan exploration paths, reuse historical results, and distill high-level insights across iterative trials without human intervention.
**Tags:** `agents`, `llm-agents`, `multi-agent`, `llm-fine-tuning`, `autonomous-research`

## [Toward Autonomous Long-Horizon Engineering for ML Research](https://arxiv.org/abs/2604.13018)
**Authors:** et al. (Submitted April 14, 2026)
**TL;DR:** Introduces an agentic framework (AiScientist) for sustained multi-hour autonomous ML research, combining hierarchical orchestration with a permission-scoped File-as-Bus workspace. A top-level Orchestrator maintains stage-level control while specialized sub-agents re-ground repeatedly on durable artifacts (plans, code, experimental evidence) to support coherent long-horizon progress.
**Tags:** `agents`, `autonomous-research`, `llm-agents`, `multi-agent`, `ml-engineering`

## [CascadeDebate: Multi-Agent Deliberation for Cost-Aware LLM Cascades](https://arxiv.org/abs/2604.12262)
**Authors:** et al. (Submitted April 14, 2026)
**TL;DR:** CascadeDebate inserts multi-agent deliberation at escalation boundaries within LLM cascade routing systems. Confidence-based routers activate lightweight agent ensembles for ambiguous queries, reaching consensus-driven resolution instead of expensively escalating to frontier models, cutting inference cost while preserving quality.
**Tags:** `agents`, `multi-agent`, `llm-cascades`, `inference-efficiency`, `deliberation`

## [EmbodiedClaw: Conversational Workflow Execution for Embodied AI Development](https://arxiv.org/abs/2604.13800)
**Authors:** et al. (Submitted April 14–15, 2026)
**TL;DR:** EmbodiedClaw provides a conversational interface for authoring, executing, and debugging embodied AI workflows through natural language. The system lowers the barrier to entry for robotic task specification by bridging high-level language instructions with low-level embodied execution pipelines.
**Tags:** `embodied-ai`, `agents`, `robotics`, `conversational-ai`, `workflow-execution`

## [When Reasoning Models Hurt Behavioral Simulation: A Solver-Sampler Mismatch in Multi-Agent LLM Negotiation](https://arxiv.org/abs/2604.11840)
**Authors:** et al. (Submitted April 12, 2026)
**TL;DR:** This paper reveals that stronger chain-of-thought reasoning can paradoxically harm multi-agent behavioral simulation by over-optimizing for strategically dominant actions and collapsing the compromise-oriented behaviors that human negotiators exhibit. It exposes a fundamental solver-sampler mismatch in reasoning-enhanced agents used to simulate bounded-rational actors.
**Tags:** `agents`, `multi-agent`, `llm-agents`, `behavioral-simulation`, `reasoning`

## [The Amazing Agent Race: Strong Tool Users, Weak Navigators](https://arxiv.org/abs/2604.10261)
**Authors:** et al. (Submitted April 11, 2026)
**TL;DR:** Introduces a benchmark using directed acyclic graph (DAG) puzzles with fork-merge tool chains that go beyond linear tool-use tasks dominating existing agent benchmarks. Results show current agents are strong individual tool users but systematically fail at multi-step navigation tasks requiring parallel branch management and result merging.
**Tags:** `agents`, `llm-agents`, `benchmarks`, `tool-use`, `evaluation`

## [Harnessing Embodied Agents: Runtime Governance for Policy-Constrained Execution](https://arxiv.org/abs/2604.07833)
**Authors:** et al. (Submitted April 10, 2026)
**TL;DR:** Proposes a runtime governance layer for embodied agents that enforces safety and policy constraints during physical task execution without requiring re-training. The framework monitors agent action sequences in real time and intercepts policy violations before they propagate to actuators.
**Tags:** `embodied-ai`, `agents`, `safety`, `robotics`, `runtime-governance`

## [A Physical Agentic Loop for Language-Guided Grasping with Execution-State Monitoring](https://arxiv.org/abs/2604.07395)
**Authors:** Wenze Wang, Mehdi Hosseinzadeh, Feras Dayoub (Submitted April 8, 2026)
**TL;DR:** Reformulates language-guided robotic grasping as a bounded embodied agent operating over grounded execution states. An agentic perception-action loop continuously monitors the execution state and replans in response to failures, enabling robust language-guided grasping in cluttered, open-world environments.
**Tags:** `embodied-ai`, `robotics`, `grasping`, `agents`, `language-grounding`

**TL;DR:** RoboPlayground is an evaluation framework that lets users author executable manipulation tasks using natural language descriptions mapped to structured physical simulation domains. It democratizes robotic benchmarking by removing the need for expert simulation engineering to define new evaluation scenarios.

## [Learning Without Losing Identity: Capability Evolution for Embodied Agents](https://arxiv.org/abs/2604.07799)
**Authors:** et al. (Submitted April 9, 2026)
**TL;DR:** Addresses the problem of catastrophic forgetting and identity drift when embodied agents acquire new capabilities through continual learning. The paper proposes a dual-memory architecture that allows capability evolution while preserving the agent's core behavioral identity across diverse task distributions.
