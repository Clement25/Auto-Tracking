# Top Papers — AI Agents & Embodied AI (2026-07-09)

## [Think Big, Search Small: Where Capacity Matters in Hierarchical Search Agents?](https://arxiv.org/abs/2607.07548)
**Authors:** (arXiv:2607.07548, July 2026)
**TL;DR:** Explores how LLM-based search agents using multi-agent architectures decompose complex questions into sub-queries dispatched to parallel sub-agents. The study investigates how model capacity should be distributed across roles in hierarchical search, finding that model size matters more in the orchestrator than in worker agents.
**Tags:** `agents`, `multi-agent`, `search`, `LLM`, `hierarchical-planning`

## [Agents with Feelings? Personality and Emotion in Multi-Agent Software Teams](https://arxiv.org/abs/2607.05659)
**Authors:** (arXiv:2607.05659, July 2026)

## [Towards Long-horizon Embodied Agents with Tool-Aligned Vision-Language-Action Models](https://arxiv.org/abs/2605.13119)
**Authors:** (arXiv:2605.13119, May 2026)

The proposed approach allows robots to call external tools (APIs, sensors, planners) within the VLA action loop, dramatically improving performance on tasks requiring 50+ steps.
