# Top Papers — AI Agents & Embodied AI (2026-06-23)

## [Beyond Entangled Planning: Task-Decoupled Planning for Long-Horizon Agents](https://arxiv.org/abs/2601.07577)
**Authors:** (arXiv:2601.07577)
**TL;DR:** Proposes a task-decoupled planning architecture that separates sub-task scheduling from execution in long-horizon agents, reducing error propagation from entangled planning pipelines. The approach improves success rates on multi-step benchmarks by allowing agents to revise individual task plans without cascading failures.
**Tags:** `agents`, `planning`, `long-horizon`, `llm`

---

## [The Evolution of Tool Use in LLM Agents: From Single-Tool Call to Multi-Tool Orchestration](https://arxiv.org/abs/2603.22862)
**Authors:** (arXiv:2603.22862)
**TL;DR:** A comprehensive survey tracing how LLM agents evolved from single API calls to complex multi-tool orchestration pipelines. Identifies key architectural patterns (sequential, parallel, hierarchical) and benchmarks the reliability tradeoffs of each approach in production environments.
**Tags:** `agents`, `tool-use`, `llm`, `survey`, `orchestration`

---

## [MCP-Zero: Dynamic Tool Discovery for LLM Agents](https://arxiv.org/abs/2506.01056)
**Authors:** (arXiv:2506.01056)
**TL;DR:** Introduces MCP-Zero, enabling LLMs to actively analyze tasks and iteratively request only the most relevant tools on demand rather than loading an entire tool registry upfront. Dynamic toolchain construction reduces context bloat and improves precision on multi-step tool-use tasks.
**Tags:** `agents`, `tool-use`, `mcp`, `llm`, `efficiency`

---

## [TodoEvolve: Learning to Architect Agent Planning Systems](https://arxiv.org/abs/2602.07839)
**Authors:** (arXiv:2602.07839)
**TL;DR:** Presents TodoEvolve, a self-improving framework where agents learn to design and refine their own planning architectures through iterative feedback loops. The system discovers effective task decomposition strategies autonomously, outperforming hand-crafted planning pipelines on complex benchmarks.
**Tags:** `agents`, `planning`, `self-improvement`, `meta-learning`

---

## [Terminal Agents Suffice for Enterprise Automation](https://arxiv.org/abs/2604.00073)
**Authors:** (arXiv:2604.00073)
**TL;DR:** Argues and demonstrates that terminal-based agents (shell-interacting LLMs) can cover the vast majority of enterprise automation use-cases without the overhead of browser or GUI agents. Provides a benchmark of 200+ real enterprise workflows showing terminal agents match or exceed GUI agents at a fraction of the compute cost.
**Tags:** `agents`, `enterprise`, `automation`, `cli`, `benchmark`

---

## [A Call for Collaborative Intelligence: Why Human-Agent Systems Should Precede AI Autonomy](https://arxiv.org/abs/2506.09420)
**Authors:** (arXiv:2506.09420)
**TL;DR:** Makes a principled case that human-in-the-loop collaborative agent systems should be the dominant paradigm before transitioning to fully autonomous AI, citing safety, alignment, and trust-building arguments. Proposes a staged autonomy ladder with concrete checkpoints for expanding agent independence.
**Tags:** `agents`, `human-ai-collaboration`, `safety`, `alignment`, `autonomy`

## [PALM: Progress-Aware Policy Learning via Affordance Reasoning for Long-Horizon Robotic Manipulation](https://arxiv.org/abs/2601.07060)
**Authors:** (arXiv:2601.07060)
**TL;DR:** PALM introduces affordance-guided progress estimation to help robots track task completion during long-horizon manipulation, enabling dynamic re-planning when sub-goals fail. Experiments on multi-step tabletop tasks show significant improvement in task completion rate over prior policy-learning baselines.
**Tags:** `embodied-ai`, `robotics`, `manipulation`, `long-horizon`, `affordance`

## [GR00T N1: An Open Foundation Model for Generalist Humanoid Robots](https://arxiv.org/abs/2503.14734)
**Authors:** (arXiv:2503.14734)
**TL;DR:** NVIDIA presents GR00T N1, an open foundation model for humanoid robots that generalizes across locomotion, manipulation, and navigation tasks from a single pretrained backbone. The model is fine-tuned with minimal task-specific data and outperforms task-specific baselines on diverse robot embodiments.

## [LA-RCS: LLM-Agent-Based Robot Control System](https://arxiv.org/abs/2505.18214)
**Authors:** (arXiv:2505.18214)
**TL;DR:** Proposes LA-RCS, a robot control architecture where an LLM agent acts as the high-level planner and coordinator, translating natural language instructions into low-level robot commands via structured intermediate representations. Demonstrated on physical robotic arms with improved instruction-following accuracy over prior NL-to-robot interfaces.
**Tags:** `embodied-ai`, `robotics`, `llm`, `agents`, `robot-control`
