## [A Survey on Autonomy-Induced Security Risks in Large Model-Based Agents](https://arxiv.org/abs/2506.23844)
**Authors:** Hang Su, Jun Luo, Chang Liu, Xiao Yang, Yichi Zhang, Yinpeng Dong, Jun Zhu
**TL;DR:** Comprehensive survey cataloguing novel security risks that emerge when LLMs gain autonomous capabilitiesŌĆömemory poisoning, tool misuse, reward hacking, and emergent misalignmentŌĆöthat go beyond threat models for conventional systems. The paper examines structural foundations of agent autonomy (long-term memory, modular tool use, recursive planning, reflective reasoning) and maps each capability to its associated attack surface. A timely reference as agentic deployments scale.
**Tags:** `agents`, `security`, `LLM`, `survey`

## [MEM: Multi-Scale Embodied Memory for Vision Language Action Models](https://arxiv.org/abs/2603.03596)
**Authors:** Marcel Torne, Karl Pertsch, Homer Walke, et al. (Google, Stanford)
**TL;DR:** Proposes a mixed-modal memory architecture for robot policies that fuses video-encoded short-horizon memory (compensating for occlusions and recent state) with text-based long-horizon semantic memory (tracking completed task stages over up to 15 minutes). Enables VLA models to handle complex multi-stage real-world tasks without sacrificing runtime latency. Demonstrated on dexterous manipulation benchmarks.
**Tags:** `embodied-ai`, `VLA`, `robotics`, `memory`, `manipulation`
