## [MIND-Skill: Quality-Guaranteed Skill Generation via Multi-Agent Induction and Deduction](https://arxiv.org/abs/2605.08670)
**Authors:** (arXiv:2605.08670)
**TL;DR:** Proposes a two-agent pipeline where an induction agent abstracts reusable skills from successful trajectories and a deduction agent reconstructs trajectories from those skills, using reconstruction, outcome, and rubric losses to guarantee skill quality. This closes the loop between skill discovery and skill validation, preventing the accumulation of noisy or redundant skills in agent libraries. Experiments show improved sample efficiency and generalization on long-horizon task benchmarks.
**Tags:** `agents`, `skill-learning`, `multi-agent`, `LLM`, `planning`

## [Beyond the Black Box: Interpretability of Agentic AI Tool Use](https://arxiv.org/abs/2605.06890)
**Authors:** (arXiv:2605.06890)
**TL;DR:** Presents a mechanistic-interpretability toolkit built on Sparse Autoencoders and linear probes that reads internal model states before each action to predict both whether a tool call is needed and how consequential the next action is likely to be. This enables real-time oversight of agentic tool-use decisions without requiring full trajectory replay. The approach offers a practical path toward auditable AI agents operating in high-stakes environments.
**Tags:** `agents`, `interpretability`, `tool-use`, `safety`, `mechanistic-interpretability`

## [MemReread: Enhancing Agentic Long-Context Reasoning via Memory-Guided Rereading](https://arxiv.org/abs/2605.10268)
**Authors:** (arXiv:2605.10268)
**TL;DR:** Addresses information loss in long-context agentic tasks by maintaining a dynamically updated memory as documents are processed linearly, combined with retrieval modules that allow agents to re-access content discarded during memory compression. The method outperforms standard long-context LLM baselines on multi-document QA and complex reasoning tasks. It demonstrates that memory architecture design is as important as raw context length for reliable agentic performance.
**Tags:** `agents`, `long-context`, `memory`, `reasoning`, `retrieval`

## [Agent Island: A Saturation- and Contamination-Resistant Benchmark from Multiagent Games](https://arxiv.org/html/2605.04312v1)
**Authors:** (arXiv:2605.04312)
**TL;DR:** Introduces a multiplayer simulation environment where language-model agents compete in cooperative and adversarial social games, designed specifically to resist benchmark saturation and data contamination that plagues static capability tests. Because the game dynamics are emergent and opponent-dependent, performance cannot be gamed by memorizing training data. Early results show significant gaps between frontier models on social reasoning and persuasion tasks.
**Tags:** `agents`, `benchmark`, `multi-agent`, `social-reasoning`, `evaluation`

## [Security Risks in Tool-Enabled AI Agents: A Systematic Analysis of Privileged Execution Environments](https://arxiv.org/html/2605.09721v1)
**Authors:** (arXiv:2605.09721)
**TL;DR:** Provides a structured taxonomy of security risks specific to cloud-hosted AI agents that can invoke tools with elevated privileges, covering prompt injection, tool misuse, credential leakage, and lateral movement. Accepted as a short paper at IEEE COMPSAC 2026, it offers a practical framework for secure agent deployment.

## [World Action Models: The Next Frontier in Embodied AI](https://arxiv.org/abs/2605.12090)
**Authors:** (arXiv:2605.12090)
**TL;DR:** Defines World Action Models (WAMs) as embodied foundation models that jointly model predictive world state and action generation, targeting a unified distribution over future states and actions rather than treating perception and control as separate modules. The paper argues that WAMs represent the natural convergence of world models and policy learning, and surveys architectural candidates, training paradigms, and open challenges. This positions WAMs as the successor framework to current VLA approaches.
**Tags:** `embodied-ai`, `world-models`, `action-models`, `foundation-models`, `robotics`

## [A Compound AI Agent for Conversational Grant Discovery](https://arxiv.org/abs/2605.02366)
**Authors:** (arXiv:2605.02366)
**TL;DR:** Unifies research funding discovery through a two-layer architecture: an aggregation layer that uses LLM-equipped browser agents to collect and normalize federal and nonprofit grant opportunities, and an agentic ReAct-based query layer that handles conversational researcher queries. The system reduces grant-discovery time by an order of magnitude compared to manual search while improving recall on niche funding opportunities. Illustrates how compound agents combining retrieval and reasoning can tackle knowledge-intensive professional tasks.
**Tags:** `agents`, `RAG`, `tool-use`, `compound-AI`, `information-retrieval`
