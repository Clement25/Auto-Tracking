# Top Papers — AI Agents & Embodied AI (2026-07-08)

## [Next-Generation Agentic Reinforcement Learning Systems Enable Self-Evolving Agents](https://arxiv.org/abs/2607.01120)
**Authors:** (arXiv:2607.01120, submitted July 1, revised July 2, 2026)
**TL;DR:** Proposes a next-generation agentic RL framework that enables LLM agents to self-evolve over time across deployment settings including coding assistants, customer-support chatbots, and scientific research assistants. The system addresses long-standing challenges of stability, sample efficiency, and continual adaptation in real-world agentic environments.
**Tags:** `agents`, `reinforcement-learning`, `self-evolving`, `LLM`

## [When Agents Do Not Stop: Uncovering Infinite Agentic Loops in LLM Agents](https://arxiv.org/abs/2607.01641)
**Authors:** (arXiv:2607.01641, submitted July 2, 2026)
**TL;DR:** Identifies and systematically studies the problem of infinite agentic loops — failure modes where LLM agents get stuck in non-terminating execution cycles. The paper proposes detection heuristics and mitigation strategies to improve agent reliability and resource efficiency.
**Tags:** `agents`, `reliability`, `LLM`, `failure-modes`

## [AgentFlow: Building Agent Dependency Graphs for Static Analysis of Agent Programs](https://arxiv.org/abs/2607.01640)
**Authors:** (arXiv:2607.01640, submitted July 2, 2026)
**TL;DR:** Introduces AgentFlow, a static analysis framework that constructs dependency graphs for multi-agent programs to identify potential issues before runtime. The approach enables developers to reason about agent interactions, data flows, and failure points without executing the system.
**Tags:** `agents`, `static-analysis`, `multi-agent`, `software-engineering`

## [AgenticSTS: A Bounded-Memory Testbed for Long-Horizon LLM Agents](https://arxiv.org/abs/2607.02255)
**Authors:** (arXiv:2607.02255, submitted July 2, 2026)
**TL;DR:** Presents AgenticSTS, a benchmark testbed specifically designed to evaluate LLM agents on long-horizon tasks under realistic bounded-memory constraints. The testbed exposes failure patterns that short-horizon evaluations miss, providing more realistic performance estimates for production deployments.
**Tags:** `agents`, `benchmark`, `long-horizon`, `evaluation`, `LLM`

## [VeriChat: An Agentic Conversational AI Assistant for Hardware Security Verification](https://arxiv.org/abs/2607.01668)
**Authors:** (arXiv:2607.01668, submitted July 2, 2026)

## [Embodied.cpp: A Portable Inference Runtime of Embodied AI Models on Heterogeneous Robots](https://arxiv.org/abs/2607.02501)
**Authors:** (arXiv:2607.02501, submitted July 2, 2026)
**TL;DR:** Presents Embodied.cpp, a portable C++ inference runtime that supports modular multi-rate execution and latency-first fused inference for embodied AI models (VLAs and world-action models) across heterogeneous robotic hardware. Evaluated on HY-VLA and pi0.5 models, it achieves 100% and 91% task success rates respectively in closed-loop control.
**Tags:** `embodied-ai`, `robotics`, `VLA`, `inference-runtime`, `edge-deployment`

## [Learning to Move Before Learning to Do: Task-Agnostic Pretraining for VLAs](https://arxiv.org/abs/2607.02466)
**Authors:** (arXiv:2607.02466, submitted July 2, 2026)
**TL;DR:** Proposes a task-agnostic pretraining stage for Vision-Language-Action (VLA) models that learns physical movement representations before task-specific fine-tuning. On a real WidowX robot, the pretrained models retain 25% task success under camera perturbations where internet-scale baselines collapse to 0%, demonstrating superior transfer and robustness.
**Tags:** `embodied-ai`, `VLA`, `robotics`, `pretraining`, `robustness`
