# Top Papers — AI Agents & Embodied AI (2026-05-23)

## [AI Agents May Always Fall for Prompt Injections](https://arxiv.org/abs/2605.17634)
**Authors:** (arXiv:2605.17634)
**TL;DR:** Prompt injection is identified as the most critical vulnerability in deployed AI agents. The paper uses Contextual Integrity theory to analyze these attacks and derives an impossibility result showing that fully defending against prompt injections is fundamentally intractable without sacrificing agent utility.
**Tags:** `agents`, `security`, `prompt-injection`, `llm`

---

## [Trust No Tool: Evaluating and Defending LLM Agents under Untrusted Tool Feedback](https://arxiv.org/abs/2605.17453)
**Authors:** (arXiv:2605.17453)
**TL;DR:** This paper addresses the growing risk of LLM agents making consequential decisions based on unverified tool outputs. It introduces an evaluation framework and defensive strategies for agents operating in environments with potentially malicious or unreliable tool feedback.
**Tags:** `agents`, `security`, `tool-use`, `llm`, `robustness`

---

## [World Action Models: The Next Frontier in Embodied AI](https://arxiv.org/abs/2605.12090)
**Authors:** (arXiv:2605.12090)
**TL;DR:** A comprehensive survey of Vision-Language-Action (VLA) models integrated with world models for embodied policy learning. The paper organizes methods into Cascaded and Joint World Action Models (WAMs) and analyzes the data ecosystem spanning robot teleoperation, human demonstrations, simulation, and egocentric video.
**Tags:** `embodied-ai`, `robotics`, `vla`, `world-models`, `survey`

---

## [Towards On-Policy Data Evolution for Visual-Native Multimodal Deep Search Agents](https://arxiv.org/abs/2605.10832)
**Authors:** (arXiv:2605.10832)
**TL;DR:** Addresses open-world multimodal deep search requiring agents to chain search, tool use, and visual reasoning over evolving context. The proposed on-policy data evolution approach improves Qwen3-VL-8B from 24.9% to 39.0% average performance across 8 multimodal deep search benchmarks.
**Tags:** `agents`, `multimodal`, `search`, `visual-reasoning`, `rl`

---

## [Agent-ValueBench: A Comprehensive Benchmark for Evaluating Agent Values](https://arxiv.org/abs/2605.10365)
**Authors:** (arXiv:2605.10365)
**TL;DR:** The first benchmark dedicated to evaluating agent values, featuring 394 executable environments across 16 domains with 4,335 value-conflict tasks. It systematically measures whether agents make ethically consistent decisions when placed in scenarios where values are in tension.
**Tags:** `agents`, `benchmark`, `alignment`, `evaluation`, `safety`

---

## [Can Agent Benchmarks Support Their Scores? Evidence-Supported Bounds for Interactive-Agent Evaluation](https://arxiv.org/abs/2605.10448)
**Authors:** (arXiv:2605.10448)
**TL;DR:** Identifies a reliability gap in interactive agent benchmarks: outcome checks based on surface-level signals cannot reliably determine whether an agent run truly succeeded. The paper introduces evidence-supported evaluation bounds that provide more rigorous guarantees.
**Tags:** `agents`, `benchmark`, `evaluation`, `methodology`

---

## [MIND-Skill: Quality-Guaranteed Skill Generation via Multi-Agent Induction and Deduction](https://arxiv.org/abs/2605.08670)
**Authors:** (arXiv:2605.08670)
**TL;DR:** A framework that automatically induces generalizable skills from successful agent trajectories using a multi-agent induction–deduction loop with quality guarantees. Skills are verified before being stored, improving reuse and agent performance on novel tasks.
**Tags:** `agents`, `skill-learning`, `multi-agent`, `llm`

---

## [Do Agents Need to Plan Step-by-Step? Rethinking Planning Horizon in Data-Centric Tool Calling](https://arxiv.org/abs/2605.08477)
**Authors:** (arXiv:2605.08477)
**TL;DR:** Compares full-horizon planning (complete plan before execution) versus single-step interleaved planning for LLM agents solving data-centric tool-calling tasks. Finds that single-step planning often matches or outperforms full-horizon approaches, challenging the assumption that long up-front plans are necessary.
**Tags:** `agents`, `planning`, `tool-use`, `llm`

---

## [LatentRAG: Latent Reasoning and Retrieval for Efficient Agentic RAG](https://arxiv.org/abs/2605.06285)
**Authors:** (arXiv:2605.06285)
**TL;DR:** Shifts retrieval-augmented generation reasoning from discrete language space to continuous latent space, producing latent tokens for thoughts and subqueries directly from hidden states in a single forward pass. Achieves performance comparable to explicit agentic RAG while significantly reducing inference latency.
**Tags:** `agents`, `rag`, `retrieval`, `efficiency`, `llm`

---

## [RoboAlign-R1: Distilled Multimodal Reward Alignment for Robot Video World Models](https://arxiv.org/abs/2605.03821)
**Authors:** (arXiv:2605.03821)
**TL;DR:** Addresses the gap between low-level training objectives and high-level task alignment in robot video world models. RoboAlign-R1 combines reward-aligned post-training with stabilized long-horizon inference, backed by a new benchmark (RobotWorldBench) of 10,000 annotated video-instruction pairs from four robot data sources.
**Tags:** `embodied-ai`, `robotics`, `world-models`, `reward-alignment`, `multimodal`

---

## [HyperEyes: Dual-Grained Efficiency-Aware Reinforcement Learning for Parallel Multimodal Search Agents](https://arxiv.org/abs/2605.07177)
**Authors:** (arXiv:2605.07177)
**TL;DR:** Presents a parallel multimodal search agent that fuses visual grounding and retrieval into a single atomic action, enabling concurrent search across multiple entities. Treats inference efficiency as a first-class training objective via dual-grained RL, improving both speed and accuracy.
**Tags:** `agents`, `multimodal`, `search`, `reinforcement-learning`, `efficiency`

---

## [3D Generation for Embodied AI and Robotic Simulation: A Survey](https://arxiv.org/abs/2604.26509)
**Authors:** (arXiv:2604.26509)
**TL;DR:** Reviews 3D generation approaches tailored for embodied AI systems that require simulation-based training. Covers how generated objects must carry kinematic structure and material properties to be useful for robot manipulation and navigation benchmarks.
**Tags:** `embodied-ai`, `robotics`, `3d-generation`, `simulation`, `survey`
