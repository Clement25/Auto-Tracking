## [Autonomous Scientific Discovery via Iterative Meta-Reflection (DiscoPER)](https://arxiv.org/abs/2607.01131)
**Authors:** Bingchen Zhao (University of Edinburgh), Sara Beery (MIT), Oisin Mac Aodha (University of Edinburgh)
**TL;DR:** DiscoPER is an autonomous LLM-powered framework for open-ended research: it dynamically generates and executes code to explore datasets without pre-specified objectives, requires all discovered patterns to pass statistical testing, and uses a second-order meta-reflection mechanism to uncover complex interconnected phenomena. Addresses the limitation of current systems that depend on constrained search spaces or predefined hypotheses.

## [VLA-Corrector: Lightweight Detect-and-Correct Inference for Adaptive Action Horizon](https://arxiv.org/abs/2607.01804)
**Authors:** Yi Pan, Miao Pan, Qi Lu, Jiaming Huang, Man Zhang, Siteng Huang, Xin Li, Jie Zhang, Yongliang Shen, Xuhong Zhang, Wenqi Zhang
**TL;DR:** Vision-Language-Action models use action chunking to predict multiple future actions open-loop, but small perturbations within this blind spot cause compounding errors in contact-rich tasks. ## [NeHMO: Neural Hamilton-Jacobi Reachability for Decentralized Safe Multi-Arm Motion Planning](https://arxiv.org/abs/2607.00326)
**Authors:** Qingyi Chen, Zachary Kingston, Ahmed H. Qureshi (Purdue University)
**TL;DR:** Safe multi-arm motion planning in high-dimensional coupled configuration spaces is hard for both centralized planners (scalability) and decentralized deep learning methods (reliance on accurate behavior prediction). NeHMO uses neural Hamilton-Jacobi Reachability learning to approximate a safety value function capturing worst-case inter-arm constraints, enabling real-time decentralized trajectory optimization without inter-arm communication or coordination protocols.
