## [Cattle Trade: A Multi-Agent Benchmark for LLM Bluffing, Bidding, and Bargaining](https://arxiv.org/abs/2605.14537)
**Authors:** (arXiv:2605.14537)
**TL;DR:** Cattle Trade is a multi-agent benchmark that evaluates LLMs as strategic agents under imperfect information, combining auctions, hidden-offer trading, bargaining, bluffing, and resource allocation challenges. It tests whether language models can reason and negotiate in adversarial, game-theoretic settings.

## [MARLIN: Multi-Agent Game-Theoretic Reinforcement Learning for Sustainable LLM Inference in Cloud Datacenters](https://arxiv.org/abs/2605.13496)
**Authors:** (arXiv:2605.13496)
**TL;DR:** MARLIN applies multi-agent game-theoretic reinforcement learning to optimize LLM inference scheduling in cloud datacenters, achieving reductions of 18% in time-to-first-token, 33% in carbon emissions, 43% in water usage, and 11% in energy costs. The framework demonstrates that agentic scheduling can substantially improve both performance and sustainability.
**Tags:** `agents`, `multi-agent`, `reinforcement-learning`, `LLM`, `green-AI`, `cloud`

## [SkillOS: Learning Skill Curation for Self-Evolving Agents](https://arxiv.org/abs/2605.06614)
**Authors:** (arXiv:2605.06614)
**TL;DR:** SkillOS is an experience-driven RL training recipe for learning skill curation in self-evolving agents, pairing a frozen executor with a trainable skill curator that updates an external skill repository from accumulated experience. The system consistently outperforms both memory-free and memory-based baselines, with the learned curator generalizing across different agent backbones and task domains.
**Tags:** `agents`, `self-evolving`, `skill-learning`, `reinforcement-learning`, `LLM`
