# Top Papers — AI Agents & Embodied AI (2026-04-17)

## [The Amazing Agent Race: Strong Tool Users, Weak Navigators](https://arxiv.org/abs/2604.10261)
**Authors:** Zae Myung Kim, Dongseok Lee, Jaehyung Kim, Vipul Raheja, Dongyeop Kang (University of Minnesota Twin Cities, Yonsei University, Google DeepMind)
**TL;DR:** Introduces AAR, a benchmark of 1,400 DAG-based puzzle instances requiring LLM agents to navigate Wikipedia and execute branching multi-step tool chains. Reveals a key gap: current agents are competent at linear tool use but struggle significantly with compositional navigation and fork-merge reasoning.
**Tags:** `agents`, `tool-use`, `benchmarks`, `LLM`, `evaluation`

---

## [EvoSpark: Endogenous Interactive Agent Societies for Unified Long-Horizon Narrative Evolution](https://arxiv.org/abs/2604.12776)
**Authors:** Shiyu He, Minchi Kuang, Mengxian Wang, Bin Hu, Tingxiang Gu (Xinjiang University, Tsinghua University)
**TL;DR:** Identifies two failure modes in long-horizon multi-agent simulations — social memory stacking and narrative-spatial dissonance — and proposes EvoSpark, a framework that sustains logically coherent narratives across extended multi-agent interactions. Enables believable emergent social dynamics in agent societies without manual intervention.
**Tags:** `multi-agent`, `LLM`, `simulation`, `long-horizon`, `narrative`

---

## [EgoNav: Learning Humanoid Navigation from Human Data](https://arxiv.org/html/2604.00416)
**Authors:** Weizhuo Wang, Yanjie Ze, C. Karen Liu, Monroe Kennedy III (Stanford University)
**TL;DR:** Trains a diffusion model entirely on 5 hours of human walking demonstrations (no robot data) to predict 6D trajectory distributions for a Unitree G1 humanoid in diverse unseen environments. Uses frozen DINOv3 visual features and a hybrid sampling scheme for real-time inference, transferring zero-shot from human to robot.
**Tags:** `embodied-ai`, `humanoid`, `navigation`, `robotics`, `diffusion-models`, `imitation-learning`

## [Modality-Native Routing in Agent-to-Agent Networks: A Multimodal A2A Protocol Extension](https://arxiv.org/html/2604.12213)
**Authors:** (arXiv:2604.12213)
**TL;DR:** Proposes a multimodal extension to the Agent-to-Agent (A2A) protocol that routes tasks natively in their original modality (text, image, audio) rather than forcing conversion through a text bottleneck. Achieves a 20-percentage-point accuracy improvement over text-bottleneck baselines on multimodal agent benchmarks.
**Tags:** `agents`, `multi-agent`, `A2A`, `multimodal`, `protocols`

---

## [From Multi-Agent to Single-Agent: When Is Skill Distillation Beneficial?](https://arxiv.org/abs/2604.01608)
**Authors:** (arXiv:2604.01608)
**TL;DR:** Empirically studies when distilling multi-agent collaboration capabilities into a single agent is advantageous, isolating the tradeoffs between coordination overhead and individual agent capacity. Finds that skill distillation is beneficial when sub-task decomposition is stable but communication latency is high.
**Tags:** `multi-agent`, `skill-distillation`, `agents`, `efficiency`

## [Multi-User Large Language Model Agents](https://arxiv.org/html/2604.08567)
**Authors:** (arXiv:2604.08567)
**TL;DR:** Proposes frameworks for LLM agents serving multiple simultaneous users with conflicting goals, addressing privacy partitioning and goal arbitration. Demonstrates that naive single-agent deployments degrade sharply under multi-user load and proposes role-scoped context management as a solution.
**Tags:** `agents`, `LLM`, `multi-user`, `privacy`, `human-AI-interaction`

## [Toward Autonomous Long-Horizon Engineering for ML Research](https://arxiv.org/html/2604.13018)
**Authors:** (arXiv:2604.13018)
**TL;DR:** Explores autonomous AI agent pipelines capable of conducting end-to-end machine learning research, from hypothesis generation through experimentation to manuscript writing. Identifies long-horizon planning and error recovery as the primary bottlenecks limiting current systems.

## [ToolTree: Efficient LLM Agent Tool Planning via Dual-Feedback Monte Carlo Tree Search and Bidirectional Pruning](https://arxiv.org/html/2603.12740)
**Authors:** (arXiv:2603.12740)
**TL;DR:** Training-free agent framework integrating an MCTS-based tool planning module with bidirectional pruning to enable robust orchestration of complex multi-tool pipelines. Outperforms state-of-the-art planning paradigms by an average of 10% success rate across diverse agentic benchmarks.
**Tags:** `agents`, `tool-use`, `planning`, `MCTS`, `LLM`

## [From LLM Reasoning to Autonomous AI Agents: A Comprehensive Review](https://arxiv.org/abs/2504.19678)
**Authors:** (arXiv:2504.19678)
**TL;DR:** Comprehensive survey tracing the evolution from chain-of-thought LLM reasoning to fully autonomous multi-agent systems, covering emerging agent communication protocols (ACP, MCP, A2A) and known failure modes including hallucination cascades and adversarial prompt injection. Offers structured recommendations for advancing secure and robust agentic AI.
**Tags:** `agents`, `survey`, `LLM`, `multi-agent`, `reasoning`, `security`

## [Humanoid-COA: Humanoid Agent via Embodied Chain-of-Action Reasoning with Multimodal Foundation Models for Zero-Shot Loco-Manipulation](https://arxiv.org/html/2504.09532v3)

## [GLANCE: A Global-Local Coordination Multi-Agent Framework for Music-Grounded Non-Linear Video Editing](https://arxiv.org/html/2604.12776)
**Authors:** (AAMAS 2026, Paphos, Cyprus)
**TL;DR:** Introduces a multi-agent system for music-driven video editing that decomposes the task between a global coordinator agent managing narrative arc and local specialist agents handling individual shot selection. Accepted at AAMAS 2026 and demonstrates significant improvements in musical coherence over single-agent baselines.
**Tags:** `multi-agent`, `video-editing`, `coordination`, `AAMAS`, `creative-AI`
