# Top Papers — AI Agents & Embodied AI (2026-05-24)

## [MOSS: Self-Evolution through Source-Level Rewriting in Autonomous Agent Systems](https://arxiv.org/abs/2605.22794)
**Authors:** Qianshu Cai et al.
**TL;DR:** MOSS enables deployed agentic systems to evolve by rewriting their own source code rather than only text-mutable artifacts like prompts or skill files. Each evolution cycle is anchored to curated production-failure evidence and verified by replaying failures against the candidate code in ephemeral trial workers before a consent-gated container swap. On the OpenClaw benchmark, MOSS lifts a four-task mean grader score from 0.25 to 0.61 in a single cycle without human intervention.
**Tags:** `agents`, `self-evolution`, `autonomous-systems`, `code-rewriting`

## [World Action Models: The Next Frontier in Embodied AI](https://arxiv.org/abs/2605.12090)
**Authors:** Siyin Wang et al.
**TL;DR:** This survey defines World Action Models (WAMs) as embodied foundation models that unify predictive state modeling with action generation in a single architecture. The authors systematically analyze the data ecosystem for WAMs — spanning robot teleoperation, portable human demonstrations, simulation, and internet-scale egocentric video — and outline open research challenges. WAMs represent a shift from task-specific policies toward general-purpose embodied intelligence.
**Tags:** `embodied-ai`, `world-models`, `robotics`, `survey`, `foundation-models`

## [SimWorld Studio: Automatic Environment Generation with Evolving Coding Agent for Embodied Agent Learning](https://arxiv.org/abs/2605.09423)
**Authors:** SimWorld Studio team
**TL;DR:** SimWorld Studio introduces SimCoder, a coding agent that self-evolves using embodied agent performance feedback to generate adaptive simulation curricula. Through co-evolution of the environment generator and the learner, the system achieves an 18-point success-rate gain over fixed-environment learning and a 40-point gain over an untrained agent in three embodied navigation case studies. This closes the costly loop between environment design and agent training.
**Tags:** `embodied-ai`, `simulation`, `curriculum-learning`, `agents`, `navigation`

## [Plan in Sandbox, Navigate in Open Worlds: Learning Physics-Grounded Abstracted Experience for Embodied Navigation](https://arxiv.org/abs/2605.10118)
**Authors:** SAGE team
**TL;DR:** SAGE (Sandbox-grounded Abstracted experience for Generalized Embodied navigation) enables agents to plan inside a physics-grounded semantic abstraction rather than a full photorealistic simulation, mimicking human mental simulation. The agent acquires transferable navigation experience at low computational cost and generalizes well to open-world environments at test time. The approach bridges the gap between abstract planning and physical grounding.
**Tags:** `embodied-ai`, `navigation`, `planning`, `simulation`, `transfer-learning`

## [Cross-Modal Navigation with Multi-Agent Reinforcement Learning](https://arxiv.org/abs/2605.06595)
**Authors:** CRONA team
**TL;DR:** CRONA proposes a Multi-Agent Reinforcement Learning framework for cross-modal embodied navigation, enabling agents to switch between or fuse different sensory modalities (e.g., vision and language) during navigation. The multi-agent design distributes modal reasoning across specialized sub-agents that communicate to produce coordinated navigation policies. Experiments demonstrate improved robustness over single-modal baselines across diverse environments.
**Tags:** `embodied-ai`, `navigation`, `multi-agent`, `reinforcement-learning`, `cross-modal`

## [Multi-Scale Gaussian-Language Map for Zero-shot Embodied Navigation and Reasoning](https://arxiv.org/abs/2605.01736)
**Authors:** GLMap team
**TL;DR:** GLMap introduces a multi-scale Gaussian-Language Map that combines explicit 3D geometry with multi-scale semantics (both instance and region concepts) and a dual-modality interface using natural language descriptions alongside 3D Gaussian representations. This enables zero-shot embodied navigation without environment-specific training by grounding language instructions in a structured spatial representation. The map supports open-vocabulary reasoning about both fine-grained objects and coarse scene regions.
**Tags:** `embodied-ai`, `navigation`, `3d-gaussian`, `zero-shot`, `vision-language`

## [AgentTrust: Runtime Safety Evaluation and Interception for AI Agent Tool Use](https://arxiv.org/abs/2605.04785)
**Authors:** AgentTrust team
**TL;DR:** AgentTrust is a runtime safety framework that intercepts potentially dangerous tool calls by AI agents before execution, combining a shell deobfuscation normalizer, SafeFix suggestions, RiskChain detection for multi-step attack chains, and a cache-aware LLM-as-Judge for ambiguous inputs. The authors release a 300-scenario benchmark across six risk categories plus 630 real-world adversarial scenarios. This addresses the critical gap between agent capability and safe deployment in production environments.
**Tags:** `agents`, `safety`, `tool-use`, `benchmark`, `security`

## [Beyond the Black Box: Interpretability of Agentic AI Tool Use](https://arxiv.org/abs/2605.06890)
**Authors:** Interpretability team
**TL;DR:** This paper introduces a mechanistic-interpretability toolkit for agentic AI that uses Sparse Autoencoders (SAEs) and linear probes to read model internal states before each tool action, inferring whether a tool is needed and estimating the consequentiality of the next action. The approach makes agent decision-making transparent enough for high-stakes enterprise workflows where black-box tool calls are unacceptable. Evaluations show the toolkit reduces unexpected tool invocations while maintaining task completion rates.
**Tags:** `agents`, `interpretability`, `tool-use`, `safety`, `mechanistic-interpretability`

## [Predictive Maps of Multi-Agent Reasoning: A Successor-Representation Spectrum for LLM Communication Topologies](https://arxiv.org/abs/2605.11453)
**Authors:** Submitted May 12, 2026
**TL;DR:** This paper provides a principled diagnostic for choosing between multi-agent LLM communication topologies (chain, star, mesh) before inference, addressing the current trial-and-error approach practitioners must use. The authors propose a successor-representation framework that maps information flow patterns onto a performance spectrum across task types. The diagnostic enables practitioners to select optimal communication topologies for new tasks without expensive empirical sweeps.
**Tags:** `multi-agent`, `llm`, `communication-topology`, `reasoning`, `systems`

## [Anticipation-VLA: Solving Long-Horizon Embodied Tasks via Anticipation-based Subgoal Generation](https://arxiv.org/abs/2605.01772)
**Authors:** Anticipation-VLA team
**TL;DR:** Anticipation-VLA addresses the failure mode of existing Vision-Language-Action models on long-horizon tasks by introducing a hierarchical architecture where an anticipation model generates adaptive subgoals that guide a VLA policy executor. Unlike prior fixed-granularity decompositions, Anticipation-VLA adjusts subgoal granularity based on execution state complexity. This significantly reduces compounding errors in multi-step manipulation and navigation benchmarks.
**Tags:** `embodied-ai`, `vision-language-action`, `robotics`, `long-horizon`, `subgoal`

## [Thinking in Text and Images: Interleaved Vision–Language Reasoning Traces for Long-Horizon Robot Manipulation](https://arxiv.org/abs/2605.00438)
**Authors:** IVLR team
**TL;DR:** IVLR (Interleaved Vision–Language Reasoning) introduces a policy framework for robotic manipulation that builds explicit intermediate representations alternating textual subgoals with visual keyframes over the full task horizon, ensuring plans are both logically coherent and geometrically grounded. This interleaving allows the robot to verify spatial feasibility at each planning step before execution. Experiments on long-horizon manipulation benchmarks show significantly higher completion rates than text-only or image-only reasoning baselines.
**Tags:** `embodied-ai`, `robotics`, `vision-language`, `manipulation`, `long-horizon`

## [Good Agentic Friends Do Not Just Give Verbal Advice: They Can Update Your Weights](https://arxiv.org/abs/2605.13839)
**Authors:** TFlow team
**TL;DR:** TFlow (Thought Flow) introduces weight-space communication between AI agents, where instead of appending messages to context, a sender agent compiles its hidden states into a transient weight perturbation applied to the receiver model. This reduces processed tokens by up to 83% and wall-clock inference time by up to 4.6× compared to text-based three-agent baselines while improving accuracy by up to 8.5 points. The approach opens a new axis for efficient multi-agent coordination beyond context-window communication.
**Tags:** `multi-agent`, `weight-space`, `efficiency`, `llm`, `communication`
