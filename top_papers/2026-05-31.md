## [Planner Matters! An Efficient and Unbalanced Multi-agent Collaboration Framework for Long-horizon Planning](https://arxiv.org/abs/2605.02168v1)
**Authors:** (arXiv:2605.02168)
**TL;DR:** Addresses LLM agents' persistent weakness in long-horizon reasoning by introducing an "unbalanced" three-role decomposition: a planner for high-level decisions, an actor for task execution, and a memory manager for contextual continuity. The asymmetric design deliberately invests more compute in planning to reduce cascading errors downstream. Demonstrates improved task completion on standard long-horizon benchmarks.
**Tags:** `agents`, `planning`, `multi-agent`, `long-horizon`
