# Top Papers — AI Agents & Embodied AI (2026-07-21)

## [JoyNexus: Service-Oriented Multi-Tenant Post-Training for VLA Models](https://arxiv.org/abs/2607.16074)
**Authors:** Yongjian Guo, Hedan Yang, Junyang Hua, Wentao Zhang, Haoran Sun, et al.
**TL;DR:** JoyNexus introduces a service-oriented multi-tenant post-training framework for Vision-Language-Action (VLA) models, decoupling training, inference, and environment services into independent layers. This architecture enables efficient sharing of VLA model post-training across multiple tenants and tasks without interference. Featured on HuggingFace Papers on July 21, 2026.
**Tags:** `vla`, `robotics`, `post-training`, `multi-tenant`, `embodied-ai`

## [Xiaomi-Robotics-1: Scaling Vision-Language-Action Models with over 100K Hours of Real-World Trajectories](https://arxiv.org/abs/2607.15330)
**Authors:** Jun Guo, Piaopiao Jin, Jason Li, Peiyan Li, Yingyan Li, Futeng Liu, Wanli Peng, et al. (Xiaomi Robotics Team)
**TL;DR:** Xiaomi presents a foundational VLA model pre-trained on over 100,000 hours of real-world manipulation trajectories collected via UMI devices — the largest real-world robotic training dataset to date. The model can follow diverse language instructions for mobile manipulation in unseen environments out-of-the-box and adapts to novel tasks with minimal fine-tuning data. A scalable auto-labeling pipeline annotates trajectory clips with natural language describing scene state transitions.
**Tags:** `vla`, `robotics`, `embodied-ai`, `manipulation`, `large-scale-training`, `dataset`

## [Towards Human-like Physical Intelligence: Lifelong Vision-Language-Action Learning for Robotic Manipulation](https://arxiv.org/abs/2607.14852)
**Authors:** Yao He, Gan Sun, Wenqi Liang, Fazeng Li, Yang Cong
**TL;DR:** LifelongVLA addresses the plasticity-stability trade-off in continual robotic learning through a dual-timescale adaptation mechanism that lets robots keep learning new tasks without forgetting old ones. A cache-efficient replay strategy enables low-cost deployment. Most lifelong learning models optimize either plasticity or stability — this work targets both simultaneously.
**Tags:** `vla`, `lifelong-learning`, `robotics`, `continual-learning`, `embodied-ai`

## [Self-Improvements in Modern Agentic Systems: A Survey](https://arxiv.org/abs/2607.13104)
**Authors:** Zhe Ren, Yimeng Chen, Dandan Guo, Guowei Rong, Tonghui Li, R. B. Xiong, Qingfeng Lan, Wenyi Wang, Li Nanbo, Yibo Yang, Mingchen Zhuge, Jürgen Schmidhuber
**TL;DR:** A comprehensive survey formalizing self-improvement in autonomous agents as a self-induced update operator that commits changes to either the foundation model parameters or the surrounding operational scaffold (prompts, memory, tools, control logic). The framework distinguishes improvements to the underlying model from improvements to its harness, and traces how self-improving agents are moving from research prototypes to deployed systems.
**Tags:** `agents`, `self-improvement`, `survey`, `llm`, `scaffolding`

## [Track, Rank, Crack: Epistemic Working Memory Scales Multi-Hop Reasoning in Language Agents (SLEUTH)](https://arxiv.org/abs/2607.12267)
**Authors:** (Not specified in search results)
**TL;DR:** SLEUTH addresses context dilution in long reasoning chains by making agent investigative state explicit via structured epistemic working memory with three components: Confirmed Facts (grounded to sources), Active Hypotheses (ranked by evidence), and Open Questions (driving next actions). This structured state replaces implicit reasoning in a growing context window, allowing multi-hop reasoning to scale without degradation.
**Tags:** `agents`, `reasoning`, `working-memory`, `multi-hop`, `llm`

## [ABot-AgentOS: A General Robotic Agent OS with Lifelong Multi-modal Memory](https://arxiv.org/abs/2607.10350)
**Authors:** (Xiaomi/Research team)
**TL;DR:** ABot-AgentOS is a deliberative agent operating system layered above low-level robot controllers, providing scene-conditioned planning, context-isolated skill execution, multi-stage verification, multi-modal memory, and edge-cloud collaboration. The paper also introduces EmbodiedWorldBench — an executable benchmark with 16 indoor/outdoor/hybrid scenes, 4 difficulty levels, and 200+ tasks covering navigation, object search, NPC dialogue, and dynamic events.
**Tags:** `embodied-ai`, `robotics`, `agent-os`, `benchmark`, `lifelong-memory`

## [Long-Horizon-Terminal-Bench](https://arxiv.org/abs/2607.08964)
**Authors:** (Not specified in search results)
**TL;DR:** A terminal-based benchmark of 46 long-horizon tasks spanning nine categories including experiment reproduction, software engineering, multimodal analysis, interactive games, and scientific computing. Unlike binary pass/fail evaluations, it enables dense intermediate rewards and partial credit so that evaluation captures not only whether an agent reaches the final goal but how far it progresses on complex, open-ended workflows.
**Tags:** `agents`, `benchmark`, `long-horizon`, `evaluation`, `software-engineering`

## [Knowledge-Centric Agents for Workflow Generation in ComfyUI](https://arxiv.org/abs/2607.15845)
**Authors:** (Not specified in search results)
**TL;DR:** This paper proposes a knowledge-centric framework for generating workflows in visual creation systems like ComfyUI, replacing brittle direct text-to-JSON generation with a principled approach. The framework performs knowledge inversion (distilling hierarchical representations from real-world workflows), knowledge injection via supervised fine-tuning, and knowledge inference that reasons from task descriptions to strategies to concrete workflow generation.
