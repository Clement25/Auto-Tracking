# Top Papers — AI Agents & Embodied AI (2026-06-17)

## [EurekAgent: Agent Environment Engineering is All You Need For Autonomous Scientific Discovery](https://arxiv.org/abs/2606.13662)
**Authors:** (EurekAgent team, multiple institutions)
**TL;DR:** EurekAgent is an environment-engineered LLM agent system for metric-driven autonomous scientific discovery. Rather than improving the agent itself, the approach focuses on building environments that amplify productive behaviors—open-ended exploration, systematic artifact management, and inter-agent collaboration—while suppressing harmful ones like reward hacking. Demonstrates that environment design alone can unlock autonomous discovery at scale.
**Tags:** `agents`, `scientific-discovery`, `llm-agents`, `multi-agent`, `reinforcement-learning`

## [The Illusion of Multi-Agent Advantage](https://arxiv.org/abs/2606.13003)
**Authors:** Prathyusha Jwalapuram et al. (10 authors)
**TL;DR:** This paper critically examines whether Multi-Agent Systems (MAS) are truly superior to Single-Agent Systems (SAS), finding that empirical claims rely on benchmarks focused on isolated reasoning tasks with weak baselines. The work calls for rigorous apples-to-apples comparisons and proposes better evaluation protocols to fairly assess when multi-agent coordination provides genuine benefits.
**Tags:** `multi-agent`, `evaluation`, `benchmarks`, `llm-agents`

---

## [CLI-Anything: Towards Agent-Native Computer Use](https://arxiv.org/abs/2606.03854)
**Authors:** (CLI-Anything team)
**TL;DR:** Proposes a paradigm shift from GUI-based computer use agents—which suffer from brittle pixel-level interactions—to CLI-native agents that interact with systems through structured command-line interfaces. The approach reduces error accumulation in long-horizon tasks and generalizes better across interface changes. Demonstrates significant gains on computer-use benchmarks.
**Tags:** `agents`, `computer-use`, `cli`, `tool-use`, `gui-agents`

---

## [MyPCBench: A Benchmark for Personally Intelligent Computer-Use Agents](https://arxiv.org/abs/2606.16748)
**Authors:** (MyPCBench team)
**TL;DR:** MyPCBench evaluates AI agents as personal computer assistants on a Linux desktop populated with 17 simulated real-world web applications and a full desktop stack. Unlike existing benchmarks that test narrow skills, MyPCBench requires agents to handle personalized, multi-app workflows mirroring real user needs. Sets a new bar for evaluating agent generalization in realistic PC environments.
**Tags:** `agents`, `computer-use`, `benchmarks`, `gui-agents`, `evaluation`

---

## [Agents' Last Exam (ALE)](https://arxiv.org/abs/2606.05405)
**Authors:** (ALE team)
**TL;DR:** ALE is a challenging benchmark designed to evaluate AI agents on long-horizon, economically valuable, real-world tasks with verifiable outcomes. It tracks how frontier models from the US and China are shifting evaluation focus from foundational capabilities toward compositional agentic tasks. Results reveal that current top models still struggle with multi-step planning over extended horizons.
**Tags:** `agents`, `benchmarks`, `evaluation`, `long-horizon`, `frontier-models`

---

## [ComAct: COM as a Unified Semantic Programmatic Interface for Computer Use Agents](https://arxiv.org/abs/2606.13239)
**Authors:** (ComAct team)
**TL;DR:** Introduces ComAct, which leverages the Component Object Model (COM) as a unified semantic programmatic interface for computer-use agents, enabling executable program synthesis and cross-application workflows. Compared to GUI agents (fragile visual grounding) and API agents (fragmented interfaces), ComAct achieves more robust, generalizable automation by operating at the semantic application layer.
**Tags:** `agents`, `computer-use`, `tool-use`, `program-synthesis`

## [An LLM System for Autonomous Variational Quantum Circuit Design](https://arxiv.org/abs/2606.13380)
**Authors:** (Multiple authors)
**TL;DR:** Demonstrates how LLM-driven agentic systems can participate in iterative scientific optimization workflows, specifically for automated variational quantum circuit design. The system autonomously explores the circuit design space, evaluates quantum performance metrics, and iterates—serving as proof-of-concept for agentic AI in specialized scientific domains.
**Tags:** `agents`, `scientific-discovery`, `quantum-computing`, `llm-agents`

## [Embodied AI Agents: Modeling the World](https://arxiv.org/abs/2506.22355)
**Authors:** (Meta AI Research)
**TL;DR:** Proposes that world models are central to reasoning and planning for embodied AI agents—including virtual avatars, wearable devices, and robots. The framework allows agents to understand and predict their environment and user intentions, enabling long-horizon planning in partially observable settings. Positions world models as the key missing ingredient for generalizable embodied intelligence.
**Tags:** `embodied-ai`, `world-models`, `robotics`, `agents`, `planning`

## [PaCo-VLA: Passivity-Shielded Compliance Prior for Contact-Rich Vision-Language-Action Manipulation](https://arxiv.org/search/?searchtype=all&query=PaCo-VLA)
**Authors:** Researchers from Peking University and JD Explore Academy
**TL;DR:** Introduces PaCo-VLA, a vision-language-action model augmented with a passivity-shielded compliance prior for safe contact-rich manipulation tasks. The approach provides formal stability guarantees during physical contact while maintaining the flexibility of learned VLA policies. Bridges the gap between safety-certified control theory and large-scale learned robot manipulation.
**Tags:** `embodied-ai`, `robotics`, `vla`, `manipulation`, `safety`

---

## [EgoInfinity: A Web-Scale 4D Hand-Object Interaction Data Engine for Robot Learning](https://arxiv.org/search/?searchtype=all&query=EgoInfinity)
**Authors:** (EgoInfinity team)
**TL;DR:** Presents EgoInfinity, a large-scale data engine that harvests 4D hand-object interaction data from web video for any-view robot retargeting and video-to-action robot learning. The system dramatically scales training data availability without robot-specific collection, enabling more generalizable manipulation policies. Demonstrates improved zero-shot transfer across diverse robot morphologies.
**Tags:** `embodied-ai`, `robotics`, `robot-learning`, `data`, `manipulation`

---

## [A Survey of Robotic Navigation and Manipulation with Physics Simulators in the Era of Embodied AI](https://arxiv.org/abs/2505.01458)
**Authors:** (Multiple institutions; last revised June 9, 2026)
**TL;DR:** Comprehensively surveys how physics simulators address the sim-to-real gap for robotic navigation and manipulation tasks in the age of foundation models. Covers recent advances in simulation fidelity, domain randomization, and transfer techniques, and benchmarks performance of foundation model-based policies across simulation-to-real transfer scenarios.
**Tags:** `embodied-ai`, `robotics`, `simulation`, `sim-to-real`, `navigation`, `manipulation`
