# Top Papers — AI Agents & Embodied AI (2026-07-18)

## [SearchOS-V1: Towards Robust Open-Domain Information-Seeking Agent Collaboration](https://arxiv.org/abs/2607.15257)
**Authors:** antins-labs team
**TL;DR:** Addresses how LLM agents fail in web search by getting trapped in repetitive loops as interaction histories grow. SearchOS is a multi-agent framework that turns fragile implicit search progress into explicit, persistent, shared state — formulating open-domain information seeking as relational schema completion with grounded citations. Introduces Search-Oriented Context Management (SOCM) with an Evidence Graph, Coverage Map, and Failure Memory.
**Tags:** `agents`, `multi-agent`, `LLM`, `search`, `RAG`, `information-retrieval`

## [SEED: Self-Evolving On-Policy Distillation for Agentic Reinforcement Learning](https://arxiv.org/abs/2607.14777)
**Authors:** Wu Jinyang, Yang Shuo, Lu Zhengxi, Zhang Fan, Shen Yuhao, et al.
**TL;DR:** Addresses the supervision gap between episode-level outcomes and token-level policy learning in outcome-based RL. SEED fine-tunes a policy to analyze completed trajectories and extract natural-language hindsight skills (reusable workflows, decisive observations, failure-avoidance rules), then re-scores sampled actions under ordinary vs. skill-augmented contexts to produce dense on-policy distillation signals jointly optimized with outcome-based RL.
**Tags:** `agents`, `reinforcement-learning`, `LLM`, `agentic-RL`, `distillation`

## [Do AI Agents Know When a Task Is Simple? Toward Complexity-Aware Reasoning and Execution](https://arxiv.org/abs/2607.13034)
**Authors:** Junjie Yin, Xinyu Feng
**TL;DR:** LLM agents default to a "maximum-context-first" strategy, re-reading files and dependencies already seen, turning a one-line edit into a full codebase audit. Formalizes minimum-sufficient execution and the Agent Cognitive Redundancy Ratio (ACRR), then proposes E3 (Estimate, Execute, Expand). On MSE-Bench (121 deterministic edits), E3 matches 100% success while cutting cost by 85%, tokens by 91%, and files inspected by 92%.
**Tags:** `agents`, `LLM`, `efficiency`, `reasoning`, `evaluation`

## [ACE-Brain-0.5: A Unified Embodied Foundational Model for Physical Agentic AI](https://arxiv.org/abs/2607.04426)
**Authors:** Ziyang Gong, Haoming Gu, Zehang Luo, Tianyi Zhang, Tao Tao, et al. (ACE-Brain Team)
**TL;DR:** Instantiates four cognitive functions in one model: Spatial Perception, Decision Making, Embodied Interaction, and Self Monitoring — realized as a closed-loop Perception-Planning-Action-Evaluation (PPAE) process. Builds on ACE-Brain-0 which used spatial intelligence as a shared scaffold, adding full physical agentic reasoning in a single unified architecture.
**Tags:** `embodied-ai`, `robotics`, `VLA`, `foundation-model`, `spatial-AI`

## [ProACT: Towards Breakdown-Aware Proactive Agent in Multi-User Collaboration](https://arxiv.org/abs/2607.03730)
**Authors:** (team not fully available)
**TL;DR:** A breakdown-aware framework for proactive agents in multi-user collaboration settings. ProACT observes speaker-attributed conversation history, detects intervention-worthy collaboration breakdowns, and chooses between silence and concise interventions via a targeted skill library. Improves appropriateness and non-interruptiveness across five LLM backbones while reducing unnecessary interruptions.
**Tags:** `agents`, `multi-agent`, `LLM`, `collaboration`, `proactive-AI`

## [Embodied.cpp: A Portable Inference Runtime of Embodied AI Models on Heterogeneous Robots](https://arxiv.org/abs/2607.02501)
**Authors:** Ling Xu, Chuyu Han, Borui Li, Hao Wu, Shiqi Jiang, Ting Cao, Chuanyou Li, Sheng Zhong, Shuai Wang
**TL;DR:** Addresses fragmentation of embodied AI model deployment across model-specific Python stacks and backend assumptions, especially on heterogeneous edge devices. Introduces a portable C++ inference runtime organized into five layers (input adapters, sequence builders, backbone execution, head plugins, deployment adapters) satisfying multi-rate closed-loop control and latency-first inference requirements for real robots.
**Tags:** `embodied-ai`, `robotics`, `VLA`, `inference`, `edge-deployment`, `systems`

## [MECoBench: A Systematic Study of Multimodal Agent Collaboration in Embodied Environments](https://arxiv.org/abs/2606.31966)
**Authors:** Qingyun Liu, Jiwen Zhang, Jingyi Hu, Siyuan Wang, Zhongyu Wei (Fudan University, CUHK, Shanghai Innovation Institute)
**TL;DR:** A multimodal embodied cooperation benchmark spanning diverse real-world tasks, two cooperation structures, and three collaboration modes. Key findings: collaboration improves task completion but depends on balancing gains against coordination overhead; communication is essential and the best mode depends on team size and model capability; collaboration improves robustness under noisy priors.
**Tags:** `embodied-ai`, `multi-agent`, `benchmark`, `MLLM`, `collaboration`

## [OmniAct: Advancing Omnimodal Embodied Agents from Isolated Skills to Everyday Physical Autonomy](https://arxiv.org/abs/2606.27251)
**Authors:** Junhao Shi, Zezheng Huai, Siyin Wang, Jia Chen, Yubang Wang, Zhaoye Fei, Hechang Chen, Jingjing Gong, Xipeng Qiu, Yu-Gang Jiang (Fudan University, Jilin University)
**TL;DR:** A hierarchical asynchronous architecture for persistent embodied agents in unstructured environments, with unified orchestration of cyber (APIs, IoT) and physical (manipulation, navigation) tools. Integrates a multimodal semantic planner, adaptive hierarchical memory with event-boundary-driven compression, and an asynchronous visual preemption engine, achieving consistent improvement across 40 real-world long-horizon tasks on two robot platforms.
**Tags:** `embodied-ai`, `robotics`, `agents`, `long-horizon`, `memory`, `IoT`

## [Embodied-R1.5: Evolving Physical Intelligence via Embodied Foundation Models](https://arxiv.org/abs/2606.11324)
**Authors:** Yifu Yuan, Yaoting Huang, Xianze Yao, Yutong Li, Pengyi Li, Zhao Zhang, Yuhao Liu, Jianye Hao, et al.
**TL;DR:** A unified Embodied Foundation Model integrating embodied cognition, task planning, correction, and pointing in a single 8B-parameter architecture. Uses three automated data construction pipelines for a 15B-token dataset and a Planner-Grounder-Corrector (PGC) closed-loop framework. Achieves SoTA on 16/24 embodied VLM benchmarks, surpassing Gemini-Robotics-ER-1.5 and GPT-5.4.
**Tags:** `embodied-ai`, `robotics`, `VLA`, `foundation-model`, `reinforcement-learning`, `long-horizon`

## [Bridging the Agent-World Gap: Text World Models for LLM-based Agents](https://arxiv.org/abs/2606.09032)
**Authors:** Yixia Li, Hongru Wang, Peng Lai, Zhiwen Ruan, He Zhu, Youxin Zhu, Ganlong Zhao, Minda Hu, Yun Chen, Sibei Yang, Peng Li, Jeff Z. Pan, Jia Pan, Guanhua Chen, Yang Liu, Guanbin Li
**TL;DR:** LLM-based agents in interactive textual environments remain largely reactive — mapping observations to actions without an explicit environment model. Proposes Text World Models (TWMs): transition models over textual states that predict the next webpage, terminal output, API response, or user reply, enabling principled planning, efficient learning, and better evaluation for web navigation and tool-use agents.
**Tags:** `agents`, `LLM`, `world-models`, `planning`, `web-agents`
