# Top Papers — AI Agents & Embodied AI (2026-07-04)

## [Harnessing the Collective Intelligence of AI Agents in the Wild for New Discoveries](https://arxiv.org/abs/2606.10402)
**Authors:** Federico Bianchi, Yongchan Kwon, Aneesh Pappu, James Zou (Together AI & Stanford)
**TL;DR:** Introduces EinsteinArena, an agent-native platform for open distributed scientific discovery where LLM agents share partial results, inspect failed attempts, and build on each other's ideas across a live set of verifiable open math problems. As of May 2026, agents on the platform have discovered 12 new state-of-the-art results surpassing all prior human and AI solutions, demonstrating that collective agent intelligence can accelerate scientific progress beyond isolated agent systems.
**Tags:** `agents`, `multi-agent`, `scientific-discovery`, `collective-intelligence`, `benchmarks`

## [Tmax: A Simple Recipe for Terminal Agents](https://arxiv.org/abs/2606.23321)
**Authors:** Hamish Ivison, Junjie Oscar Yin, Rulin Shao, Teng Xiao, Nathan Lambert, Hannaneh Hajishirzi (Allen AI)
**TL;DR:** Presents Tmax, the strongest open RL training recipe for terminal-using LLM agents, achieving 27% on Terminal-Bench 2.0 with only a 9B-parameter model — outperforming much larger closed models. The recipe introduces a novel data taxonomy combining difficulty control, diverse personas, and verifier diversification, releasing both 9B and 27B model weights publicly to push the open-source frontier for agentic coding.
**Tags:** `agents`, `rl-training`, `terminal-agents`, `coding`, `open-source`

## [From Chatbot to Digital Colleague: The Paradigm Shift Toward Persistent Autonomous AI](https://arxiv.org/abs/2606.14502)
**Authors:** Yongheng Zhang, Ziang Liu, Jiaxuan Zhu, Shuai Wang, Xiangqi Chen, et al.
**TL;DR:** Provides a comprehensive framework mapping the evolution of LLMs from stateless conversational generators into "Digital Colleagues" — persistent systems capable of long-horizon reasoning, tool-augmented task execution, memory, and self-improvement. The paper articulates two key transitions: from fast-thinking chatbots to deliberate Thinking LLMs, and from ad-hoc tool-calling agents to workstation-style systems with persistent workspaces, skill libraries, and governance loops.
**Tags:** `agents`, `survey`, `persistent-ai`, `autonomy`, `llm`

## [Multi-Agent Transactive Memory](https://arxiv.org/abs/2606.19911)
**Authors:** To Eun Kim, Xuhong He, Dishank Jain, Ambuj Agrawal, Negar Arabzadeh, Fernando Diaz
**TL;DR:** Proposes Multi-Agent Transactive Memory (MATM), a framework for population-level storage and retrieval of agent-generated trajectories so that newly instantiated agents can leverage procedural knowledge discovered by prior agents rather than rediscovering solutions from scratch. MATM extends retrieval-augmented generation to heterogeneous multi-agent populations, acting as a "search engine" for agent-produced artifacts, reducing redundant computation at scale.
**Tags:** `multi-agent`, `memory`, `retrieval-augmented-generation`, `agent-infrastructure`

## [RaMem: Contextual Reinstatement for Long-term Agentic Memory](https://arxiv.org/abs/2606.22844)
**Authors:** Wei Yang, Bryce Kan, Shixuan Li, Li Li, Yuehan Qin, Jiate Li, Paul Bogdan, Jesse Thomason
**TL;DR:** Identifies "context collapse" as a core failure mode in long-term agent memory — where memory fragments from different situations appear equally relevant during retrieval because surrounding context is lost during compression. Proposes RaMem (Contextual Reinstatement for Agentic Memory), a framework that converts retrieved fragments into contextually verifiable evidence, significantly improving memory fidelity in extended agent interactions.
**Tags:** `agents`, `memory`, `long-term-memory`, `retrieval`, `context`

## [State-Grounded Multi-Agent Synthetic Data Generation for Tool-Augmented LLMs](https://arxiv.org/abs/2606.16307)
**Authors:** Rahul Khedar, Eshita, Sneha Teja Sree Reddy Thondapu, et al. (PayPal AI)
**TL;DR:** Introduces StateGen, a synthetic data generation platform that produces reasoning-trace-rich training data for tool-augmented LLM agents by orchestrating a four-role LLM loop (user simulator, agent, state-grounded tool simulator, multi-axis judge). The key innovation is an authoritative state manager that enforces a backend-is-truth invariant across turns, eliminating the dominant class of tool-call hallucinations by construction and extending naturally to hierarchical multi-agent settings.
**Tags:** `agents`, `synthetic-data`, `tool-use`, `multi-agent`, `training`

## [Embodied-R1.5: Evolving Physical Intelligence via Embodied Foundation Models](https://arxiv.org/abs/2606.11324)
**Authors:** Yifu Yuan, Yaoting Huang, Xianze Yao, Yutong Li, Shuoheng Zhang, Linqi Han, Pengyi Li, et al.
**TL;DR:** Presents a unified 8B-parameter Embodied Foundation Model integrating cognition, task planning, correction, and pointing within a single architecture via a Planner-Grounder-Corrector (PGC) closed-loop framework. Trained on a 15B-token data system built from three automated pipelines and a multi-task balanced RL recipe, Embodied-R1.5 achieves state-of-the-art results on 16 of 24 embodied VLM benchmarks, surpassing Gemini-Robotics-ER-1.5 and GPT-5.4 despite its smaller size.
**Tags:** `embodied-ai`, `robotics`, `foundation-models`, `vision-language-action`, `reinforcement-learning`

## [Hy-Embodied-0.5-VLA: From Vision-Language-Action Models to a Real-World Robot Learning Stack](https://arxiv.org/abs/2606.14409)
**Authors:** He Zhang, Lingzhu Xiang, Haitao Lin, Zeyu Huang, Minghui Wang, et al. (Tencent Robotics X & Hunyuan Team)
**TL;DR:** Presents HyVLA-0.5, an end-to-end robot learning system from Tencent spanning the full stack: data collection, model design, continued pre-training, SFT, RL post-training, and real-world deployment. The paper provides a detailed blueprint for building production-grade Vision-Language-Action models, including infrastructure lessons learned deploying VLAs on physical robots in real environments.
**Tags:** `embodied-ai`, `robotics`, `vision-language-action`, `robot-learning`, `vla`

## [Safe Embodied AI for Long-horizon Tasks: A Cross-layer Analysis of Robotic Manipulation](https://arxiv.org/abs/2606.05660)
**Authors:** Dabin Kim, Daemin Park, Sangyub Lee, Jinsik Kim, Yeongtak Oh, Jongho Shin, Sungroh Yoon
**TL;DR:** Provides a structured survey of safety in long-horizon robotic manipulation organized by intervention locus — covering planning-time, policy-time, and execution-time safety mechanisms. The paper analyzes the strength of evidence for each approach (formal guarantees, statistical support, empirical heuristics) and identifies how semantic misgrounding, subtask-level error propagation, execution drift, and contact-rich physical risk accumulate in closed-loop embodied systems.
**Tags:** `embodied-ai`, `safety`, `robotics`, `manipulation`, `survey`

## [A Technical Taxonomy of LLM Agent Communication Protocols](https://arxiv.org/abs/2606.19135)
**Authors:** (Multiple authors — Jun 2026)
**TL;DR:** As multi-agent LLM systems scale, robust inter-agent communication protocols have become critical infrastructure. This paper proposes a technical taxonomy classifying existing and emerging agent communication protocols by structure, semantics, and coordination patterns, providing a reference framework for designing interoperable distributed agent networks. The work fills a gap between ad-hoc protocol choices in current systems and the principled infrastructure needed for production-grade multi-agent deployments.
**Tags:** `agents`, `multi-agent`, `protocols`, `communication`, `infrastructure`
