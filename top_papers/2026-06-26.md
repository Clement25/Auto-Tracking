# Top Papers — AI Agents & Embodied AI (2026-06-26)

## [FastContext: Training Efficient Repository Explorer for Coding Agents](https://arxiv.org/abs/2606.14066)
**Authors:** Shaoqiu Zhang
**TL;DR:** FastContext introduces a dedicated exploration subagent that decouples repository navigation from code solving in LLM-based coding agents. It issues parallel tool calls to return concise file paths and line ranges as focused context, dramatically reducing noise and improving task accuracy in large codebases.
**Tags:** `agents`, `coding-agents`, `llm`, `software-engineering`, `context-retrieval`

## [Safe Embodied AI for Long-horizon Tasks: A Cross-layer Analysis of Robotic Manipulation](https://arxiv.org/abs/2606.05660)
**Authors:** (SAE World Congress 2026 contributors)
**TL;DR:** This paper presents a cross-layer safety analysis of embodied AI systems performing long-horizon manipulation tasks, covering planning-time, policy-time, and execution-time safety interventions. It argues that safety must be treated as a systems-level property rather than a single-layer concern, providing a framework applicable across diverse robotic platforms.
**Tags:** `embodied-ai`, `robotics`, `safety`, `manipulation`, `long-horizon`

## [The Evolution of Tool Use in LLM Agents: From Single-Tool Call to Multi-Tool Orchestration](https://arxiv.org/abs/2603.22862)
**Authors:** (Multi-institution survey)
**TL;DR:** This comprehensive survey traces how tool use in LLM agents has evolved from simple single-call invocations to complex multi-tool orchestration over long trajectories. It organizes the field around six core dimensions: inference-time planning, training and trajectory construction, safety and control, efficiency, capability completeness, and benchmark design.
**Tags:** `agents`, `tool-use`, `llm`, `survey`, `multi-tool`, `orchestration`

## [ARIS: Open-Source Research Harness for Cross-Model Adversarial Collaboration](https://arxiv.org/abs/2503.25723)
**Authors:** Shanghai Jiao Tong University
**TL;DR:** ARIS is an open-source multi-agent research harness built on cross-model adversarial collaboration, using coordinated execution, orchestration, and assurance layers to ensure reliable long-term research outcomes. It provides a reproducible framework for running extended research pipelines where multiple LLMs challenge and verify each other's outputs.

## [Human-like Autonomy Emerges from Self-play and a Pinch of Human Data](https://arxiv.org/list/cs.MA/current)
**Authors:** Daphne Cornelisse, Julian Hunt, Zixu Zhang, Waël Doulazmi, Kevin Joseph, Jaime Fernández Fisac
**TL;DR:** This paper demonstrates that rich human-like autonomous behavior in agents can emerge primarily from self-play reinforcement learning augmented with only a small amount of human demonstration data. The approach significantly reduces the dependency on costly human annotations while achieving behavior that closely mirrors human decision-making patterns.
**Tags:** `agents`, `self-play`, `reinforcement-learning`, `human-ai`, `autonomy`

## [MemSlides: Hierarchical Memory Framework for Personalized Presentation Agents](https://arxiv.org/abs/2508.08645)
**Authors:** (EMNLP 2026 ARR submission)
**TL;DR:** MemSlides introduces a three-tier hierarchical memory system for presentation-generation agents, separating long-term user profiles, working session memory, and reusable tool-execution memory. This architecture enables stable personalization and reliable local edits across multi-turn revision workflows, outperforming flat-memory baselines.
**Tags:** `agents`, `memory`, `personalization`, `presentation`, `multi-turn`

## [Emergent Collaborative Deliberation in Multi-Model AI Systems: A BFT-Derived Protocol for Epistemic Synthesis](https://arxiv.org/list/cs.MA/current)
**Authors:** (Multi-institution)
**TL;DR:** This paper proposes a Byzantine Fault Tolerant-inspired protocol for achieving epistemic consensus across heterogeneous LLM agents, enabling collaborative deliberation even when individual models disagree or produce erroneous outputs. It demonstrates that multi-model systems using this protocol achieve higher factual accuracy than any single constituent model.
**Tags:** `multi-agent`, `consensus`, `bft`, `epistemic-reasoning`, `robustness`

## [ClawsBench: Evaluating Capability and Safety of LLM Productivity Agents in Simulated Workspaces](https://arxiv.org/abs/2604.05172)
**Authors:** (Benchmark Research Team)
**TL;DR:** ClawsBench creates a rich simulated workplace environment to evaluate LLM agents across productivity tasks while simultaneously probing for safety violations such as unauthorized data access and policy circumvention. It reveals that high task-completion performance and safety compliance are often in tension, motivating new training approaches.
**Tags:** `agents`, `benchmark`, `safety`, `productivity`, `evaluation`
