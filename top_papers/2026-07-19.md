## [RxBrain: Embodied Cognition Foundation Model with Joint Language-Visual Reasoning and Imagination](https://arxiv.org/abs/2607.14187)
**Authors:** Haotian Liang, Mingkang Chen, Yufei Huang, Yuchun Guo, Xiaomeng Zhu, Xiangli Shi, Kaixuan Wang, Yunxuan Mao, Weijie Zhou, Ling Chen, et al. (Tencent Hunyuan)
**TL;DR:** Introduces Hy-Embodied-RxBrain, an embodied cognition model that plans in a unified sequence where language reasoning and visual imagination play complementary roles — unlike models that treat these as separate passes. The key innovation is joint language-visual planning: the model produces both textual decisions and visual imaginations of future states in a single generation. Addresses a fundamental gap between VLMs that reason in text and world models that predict visual states, without bridging them.
**Tags:** `embodied-ai`, `robotics`, `foundation-model`, `vision-language`, `planning`, `world-model`

## [ACE-Brain-0.5: A Unified Embodied Foundational Model for Physical Agentic AI](https://arxiv.org/abs/2607.04426)
**Authors:** Ziyang Gong, Haoming Gu, Zehang Luo, Tianyi Zhang, Tao Tao, Yixiao Chi, Zhe Liu, Lingsi Zhu, Jingyuan Liu, Anke Tang, et al. (Brain Team, 30 authors)
**TL;DR:** ACE-Brain-0.5 extends its predecessor from a perception-centric model into a full closed-loop embodied agent that can perceive, plan, act, monitor, and self-improve. Uses a single 8B all-in-one mixture-of-transformer backbone with task-specific decoders for language, spatial predictions, and continuous robot actions. Organizes robot intelligence into five coupled cognitive functions: Spatial Perception, Decision Making, Embodied Interaction, Self Monitoring, and Self Improvement.
**Tags:** `embodied-ai`, `robotics`, `foundation-model`, `agents`, `manipulation`, `navigation`, `self-improvement`

## [Embodied-R1.5: Evolving Physical Intelligence via Embodied Foundation Models](https://arxiv.org/abs/2606.11324)
**Authors:** Yifu Yuan, Yaoting Huang, Xianze Yao, Yutong Li, Shuoheng Zhang, Linqi Han, Pengyi Li, Jiangeng Sun, Wenting Jia, Zhao Zhang, et al. (Tianjin University, Tencent Hunyuan)
**TL;DR:** A unified Embodied Foundation Model integrating cognition, task planning, correction, and pointing in a single 8B-parameter architecture. Built on 15B+ tokens of automatically constructed training data and a Planner-Grounder-Corrector (PGC) closed-loop framework for autonomous self-correction over long-horizon tasks. Achieves SOTA on 16 of 24 embodied VLM benchmarks, surpassing Gemini-Robotics-ER-1.5 and GPT-5.4, and can be fine-tuned into a VLA outperforming π0.5 across 4 manipulation suites.
**Tags:** `embodied-ai`, `robotics`, `foundation-model`, `vla`, `manipulation`, `long-horizon`, `rl`

## [MECoBench: A Systematic Study of Multimodal Agent Collaboration in Embodied Environments](https://arxiv.org/abs/2606.31966)
**Authors:** Qingyun Liu, Jiwen Zhang, Jingyi Hu, Siyuan Wang, Zhongyu Wei (Fudan University, Shanghai Innovation Institute, CUHK)
**TL;DR:** Introduces MECoBench, a benchmark for evaluating how multimodal large language models collaborate as embodied agents in visually grounded environments. Studies two cooperation structures and three collaboration modes across diverse real-world tasks. Key findings: collaboration improves task completion but depends critically on communication quality; the optimal collaboration mode varies by team size and model capability; collaboration improves robustness under noisy priors.
**Tags:** `embodied-ai`, `multi-agent`, `benchmark`, `collaboration`, `multimodal`, `vision-language`

## [When Multi-Robot Systems Meet Agentic AI: Towards Embodied Collective Intelligence](https://arxiv.org/abs/2606.27929)
**Authors:** Yuxuan Yan, Yuanyuan Jia, Qianqian Yang
**TL;DR:** Proposes Embodied Collective Intelligence (ECI) as a future multi-robot paradigm where robot teams accumulate and share world context, task progress, and skill experience as collective resources. Presents ECI through three pillars — Co-Perception, Co-Action, and Co-Evolution — and demonstrates shared world-memory inheritance in a navigation study. Bridges the gap between single-robot autonomy and distributed agentic multi-robot systems that can leverage heterogeneous capabilities and fault tolerance.
**Tags:** `embodied-ai`, `multi-agent`, `robotics`, `collective-intelligence`, `multi-robot`, `shared-memory`

## [Qwen-AgentWorld: Language World Models for General Agents](https://arxiv.org/abs/2606.24597)
**Authors:** Qwen Team (Alibaba)
**TL;DR:** Introduces native language world models trained via a three-stage pipeline (Continual Pre-Training, SFT, RL) for simulating agentic environments across seven distinct domains. The world model can serve either as a decoupled environment simulator for agent training, or as a unified agent-plus-world foundation model. Addresses the bottleneck of requiring physical or code environments for agent training by enabling fully language-based simulation.
**Tags:** `agents`, `world-model`, `language-model`, `rl`, `simulation`, `multi-domain`

## [Are We Ready For An Agent-Native Memory System?](https://arxiv.org/abs/2606.24775)
**Authors:** (Research team — full list via arXiv)
**TL;DR:** Proposes a systematic framework decomposing agent memory into four dimensions: representation, extraction, retrieval, and maintenance. ## [SAFARI: Scaling Long Horizon Agentic Fault Attribution and Recovery](https://arxiv.org/abs/2606.24626)
**Authors:** (Research team — full list via arXiv)
**TL;DR:** Introduces SAFARI, a tool-augmented diagnostic loop that replaces linear context loading for trajectory fault analysis in long-horizon agent tasks. Instead of loading full trajectories into context, SAFARI uses structured fault attribution to identify exactly where and why agent execution diverged, enabling targeted recovery without context explosion. Addresses a major scalability bottleneck in debugging agentic systems operating over hundreds of steps.
**Tags:** `agents`, `fault-attribution`, `long-horizon`, `reliability`, `debugging`, `scalability`

## [Governed Shared Memory for Multi-Agent LLM Systems](https://arxiv.org/abs/2606.24535)
**Authors:** (Research team — full list via arXiv)
**TL;DR:** Presents MemClaw, an architecture for governed shared memory in multi-agent LLM systems that addresses access scoping and temporal correctness — two critical correctness properties missing from existing shared memory approaches. Provides formal guarantees about which agents can read/write which memory at which points in a workflow, preventing a class of subtle multi-agent bugs caused by uncontrolled memory access patterns.
**Tags:** `multi-agent`, `memory`, `architecture`, `formal-methods`, `agents`, `correctness`

## [NatureBench: AI Coding Agents vs Published State-of-the-Art](https://arxiv.org/abs/2606.24530)
**Authors:** (Research team — full list via arXiv)
**TL;DR:** Introduces NatureBench, a cross-discipline benchmark of 90 tasks drawn from Nature-family publications, testing whether AI coding agents can replicate or advance published scientific computing results. Provides a rigorous evaluation of agent capability on high-complexity, open-ended scientific tasks rather than curated toy benchmarks. Finds significant gaps between agent performance on standard coding benchmarks and real scientific replication tasks.
**Tags:** `agents`, `coding`, `benchmark`, `scientific-computing`, `evaluation`, `code-generation`

## [Embodied AI Agents: Modeling the World](https://arxiv.org/abs/2506.22355)
**Authors:** (Research survey — full list via arXiv)
**TL;DR:** Comprehensive survey positioning world models as the central mechanism for reasoning and planning in embodied AI agents — whether virtual avatars, wearable devices, or physical robots. Covers how agents perceive, learn from, and predict within their environments using world-model-based planning, and maps the landscape of approaches across simulation-to-real transfer, spatial reasoning, and goal-conditioned behavior. Essential reference for the embodied AI research agenda.
**Tags:** `embodied-ai`, `world-model`, `survey`, `robotics`, `planning`, `agents`
